# 🚀 Video Call App (Zoom like app)

<p align="center">
  <b>Real-time video conferencing web app with chat, screen sharing & responsive UI</b>
</p>

---

## 🌐 Live Demo

🔗 https://video-call-app-ten-eosin.vercel.app/

---

## ✨ Features

* 🎥 Real-time Video Calling (WebRTC)
* 🎤 Mute / Unmute Audio
* 📷 Camera On / Off
* 🖥️ Screen Sharing
* 💬 Live Chat System
* 🔔 Unread Message Notification
* 👥 Multi-user Support
* 📱 Fully Responsive (Mobile + Desktop)
* 🎨 Modern UI (Glassmorphism Design)

---

## 🛠️ Tech Stack

### Frontend

* ⚛️ React.js
* 🎨 CSS (Responsive Design)
* 🧩 Material UI

### Backend

* 🟢 Node.js
* 🚀 Express.js
* 🔌 Socket.io
* 📡 WebRTC

### Database

* 🍃 MongoDB

## ⚙️ Installation Guide

### 1️⃣ Clone Repo

```bash
git clone https://github.com/Subhamsahoo94/video-call-app
cd video-call-app
```

---

### 2️⃣ Install Dependencies

#### Frontend

```bash
cd frontend
npm install
npm start
```

#### Backend

```bash
cd backend
npm install
node src/app.js
```

---

## 🔐 Environment Variables

Create `.env` file in backend:

```env
MONGO_URI=your_mongodb_connection_string
```
